Program Structure & Build Pipeline

#include <stdio.h>          // preprocessor: paste header
int add(int x, int y);      // forward declaration (prototype)

int main(int argc, char *argv[]) { // entry point
    printf("%d", add(1, 34));
    return 0;                // 0 = success, non-zero = error
}

int add(int x, int y) { return x + y; } // definition

Pipeline: Source(.c) → Preprocessor (gcc -E) → .i → Compiler (gcc -S) → .s → Assembler (gcc -c) → .o → Linker (gcc) → executable

One-shot build: gcc file.c -o program

Preprocessor Directives

DirectiveSemantics
#include <x.h>System header (standard paths)
#include "x.h"Local header (current dir first)
#define NAME valObject-like macro (text substitution)
#define f(x) (expr)Function-like macro (inline substitution)
#define FLAGFlag macro, value-less (conditional compilation)
#ifdef / #ifndefConditional guards, close with #endif

Header guard pattern — avoids double-inclusion:

#ifndef CUSTOM_HEADER_H
#define CUSTOM_HEADER_H
// declarations / prototypes go here
#endif

RuleDirectives end with a newline, never a semicolon.

Objects, Variables & Core I/O

Object: region of storage holding a value. Variable: a named object.

sizeof(x) is a compile-time operator, not a runtime function call.

Standard streams: stdin (keyboard), stdout (console), stderr (errors → fprintf(stderr, ...)).

printf("%d %.2f %s\n", age, gpa, name); // pass by value
scanf("%d %f", &age, &salary);          // needs address (&)
fgets(buf, sizeof(buf), stdin);         // safe full-line read

WarningNever use gets() (no bounds checking). Use fgets().

FunctionDirectionNeeds &Returns
printfOutputNoChars written

Initialization & Identifiers

Allowed styles:

int a = 2; — copy init, C's only style.
int a(10); ✗ C++
int a{10}; ✗ C++

UBLocal vars hold garbage if uninitialized. Global/static vars auto-init to 0 / NULL.

Naming: no keywords; letters/digits/_ only; can't start with a digit; case-sensitive (age != Age).

Keywords, Expressions & Scope

No OOP: C has no class, this, new, delete, public, private, virtual.

Qualifiers: const (read-only), restrict (sole-alias optimization hint), volatile (may change outside program control).

IdentifierScopeLifetime / linkage
Local variableDeclaration → end of blockAutomatic (stack)
Function parameterWhole function bodyAutomatic (stack copy)
Global variableDeclaration → end of fileStatic storage, auto-init 0
static local varScoped to functionPersists across calls

extern int g_val; references a global defined elsewhere. static on a global/function gives internal linkage — hidden from other files.

Strings & Pointers

C strings are 1D char arrays terminated by a mandatory null byte ('\0').

strncpy(dest, src, sizeof(dest) - 1);
dest[sizeof(dest) - 1] = '\0'; // force null-termination
if (strcmp(a, b) == 0) { ... } // never use == on strings
SyntaxPointerData
char* sMutableMutable
const char* sMutableRead-only
char* const sFixedMutable
const char* const sFixedRead-only
int *a, b;  // trap: 'a' is int*, 'b' is plain int!
int *a, *b; // correct way to declare two pointers
int (*op)(int, int) = add; // function pointer

Flow Control, Loops & Bases

Loop types:

  • while (c) — checks first, min runs 0
  • do {} while (c); — checks last, min runs 1
  • for (i; c; s) — checks first, min runs 0

break exits the loop/switch. continue jumps to the next check.

BasePrefixFormat
Octal0%o
Hex0x%x / %X
Binary0b%b C23

The # flag (e.g. %#X) prefixes automatically.

Ruleswitch(expr) needs an integral type (int, char, bool, enum). Always break; unless you intend to fall through.
